# Event-Registration-Platform
A fully functional e-commerce shopping cart system with real-time updates and checkout flow.
A simple web-based platform that allows users to view available events and register online easily. It also helps organizers manage event details and participant information in a more organized way.

### Features
View available events
Read event details
Register for events online
Manage participant information
Simple and user-friendly interface

### Technologies Used
HTML
CSS

### How to Run
Download or clone the project files.
Open the project folder.
Open index.html in your browser.
